# google-meridian
Testing the features of Google's MMM solution

# About Meridian
Meridian is Google's open source MMM solution based in Python. More information about this tool can be found here <https://developers.google.com/meridian/docs/basics/meridian-introduction>[Google Meridian Introduction].

As a professional working in marketing measurement, this repo contains the demo test of Google Meridian using the synthetic data provided by Google. It contains exploratory data analysis to understand different marketing spends, building the model, setting priors, and finally follows by setting optimal budget allocations for maximizing marketing ROI/ROAS.
